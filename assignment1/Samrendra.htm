Hypertext Markup Language

Grappling from website to website in a seemingly ironic yet endless journey to understand how websites themselves work has lead me here.
Hypertext Markup Language (HTML) is the standard markup language for creating web pages and web applications. With Cascading Style Sheets (CSS) and JavaScript, it forms a triad of cornerstone technologies for the World Wide Web"

HTML was first published in 1995 as RFC 1866 and ever since it has been continually developed and enhanced.

  • HTML 2.0 1995
  • HTML 3.2 1997
  • HTML 4.0 1997
  • HTML 4.01 1991
  • HTML5 2014
  • HTML5.1 2016
  • HTML5.2 2017
